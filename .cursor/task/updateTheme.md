# Edite o tema
Com base no layout da home em /layout/home. Edite o arquivo de tema em: src\themes\custom-theme.scss atualizando as variaveis e seguindo as orientações abaixo:
1. Atualize para que as variaveis de cores primárias e secundárias façam sentido com o layout
2. Atualize a fontFamily com base no layout
3. Atualize o container da loja: largura , espaçamento interno e margins
